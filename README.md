# AI Travel Sales Assistant (Streamlit Cloud MVP Setup)

travel

This repository uses real-world Kaggle datasets to drive conversational RAG experiences.

### Deployment instructions on Streamlit Cloud:
1. Push this complete collection of files into a clean public/private GitHub repository.
2. Log in directly to [share.streamlit.io](https://share.streamlit.io).
3. Connect the repository, set the branch to `main`, and define the main file path as `streamlit_app.py`.
4. Click **Deploy**. `kagglehub` will download your runtime files automatically when initialized.
